Welcome to the Python User Group in Princeton

The Python User Group in Princeton (PUG-IP) is a friendly and diverse group of programmers of all levels and backgrounds, bound together by a passion for elegant code and an interest in Python as one way to achieve that goal. The group participates in and hosts various educational and community outreach events.

Meetings

Our meetups are held on the second Monday of each month at the Princeton Public Library. Please see the PUG-IP meetup.com page for details about upcoming meetups.

Other Events

In February 2012, volunteers from PUG-IP taught an introductory Python course that was very well received. Stay tuned to this site and the Princeton Public Library Facebook page for announcements about future events.

This Site

The pugip.org site is a collaborative effort among any members who care to be involved. It's a way for members to stretch their wings and work with things we might not otherwise get our hands on. It's fun! Stay tuned here for future developments. We collaborate via a github organization, and deploy to Heroku.
